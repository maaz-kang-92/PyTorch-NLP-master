torchnlp.samplers package
=========================

The ``torchnlp.samplers`` package introduces a set of samplers. Samplers sample elements from a 
dataset. ``torchnlp.samplers`` plug into ``torch.utils.data.distributed.DistributedSampler`` and
``torch.utils.data.DataLoader``.
